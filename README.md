# Function Carrying Visualizer (React + TypeScript + Vite + Tailwind)

### Stack

- React
- TypeScript
- TailwindCSS
- Vite + SWC

## Pre-requisites

- Node `>=20.10`

## Commands

1. `npm install` - Install all dependencies
2. `npm run dev` - Run the local development server
3. `npm run build` - Create a production build
